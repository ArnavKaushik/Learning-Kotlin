An Introduction to Kotlin

By Arnav Kaushik

The Basics

Main

Main is an entry point for a Kotlin. It is used to begin program execution.

fun main() {
    println("Hello World")
}

Printing

Printing in Kotlin will print to the console of your IDE. There are two main print functions:

print()
println()

print will print on the same line as the last line printed, and println will print a new line. A println followed by a print will print the second statement on a new line, since the last line printed was a new line.

Variables

To create a variable, use the keyword var. A variable can be mutated and assigned a value.

var helloWorld = "Hello World"
helloWorld = "Hello World!"

Types of data that can be assigned to a variable include:

  • Strings
  • Integers
  • Floats
  • Doubles
  • Objects

Constants

To create a constant, use the keyword val. A constant can NOT be altered, and will contain the value at its initialization.

val helloWorld = "Hello World"
//helloWorld = "A" will not work

It is read only after being initialized.

Functions

To create a function, use the keyword fun. A function will execute the code in its body and may return a value. It can also accept arguments.

fun sayHelloWorld() {
    println("Hello World")
}

To return something from a function, use the : operator following the function.

fun sayHelloWorld(): String {
    return "Hello World"
}

To pass arguments into the function, use the parameters inside the parenthesis.

fun sayHelloWorld(name: String) {
    println("Hello World ${name}")
}

Functions can be written with an expression body and an inferred return type.

fun sum(a: Int, b: Int) = a + b

Comments

Comments can be used to mark up code. Anything in a comment will NOT be executed.

//Single Line Comment

/*
Multiline
Comment
*/

/*
Comments
/*can also be nested*/
like this
*/

String templates

Strings can contain other variables and values. This can be done using $.

var a = 2
val s1 = "$a" 
println(s1) // "2"

You can also place expressions inside of the brackets.

var a = 1
val s1 = "${a + 2}"
println(s1) //"3"

Conditional expression

Conditional expressions evaluate a condition and executes the code corresponding to that condition.

int a = 1
int b = 2
if ( a < b ) {
    println(b)
} else {
    println(a)
}

if statements can also be used as an expression.

fun maxOf(int a, int b) = if (a > b) a else b

Null values and optionals

Null values (null) are values that do not have a value. This is denoted with a ?.

For example, let's say we have a function called parseInt that returns an Int?. 

    fun parseInt(str: String): Int? {
        //...
    }

Let's create a function called printProduct that prints the product of two strings if they are Int.

fun printProduct(arg1: String, arg2: String) {
    val x = parseInt(arg1) //Int?
    val y = parseInt(arg2) //Int?

    if (x != null && y != null) {
        //Because both x and y have been null checked (!= null), they are automatically casted to non-nullable
        println(x * y)
    } else {
        println("The arguments passed are not integers.")
    }

}

Type Checks

The is operator can check if an expression is an instance of a type. If true, then the expression is automatically casted. If not, it remains the type Any.

if (obj is String) {
    return obj.length
}

Control loops

Control Loops are ways to execute code for a certain amount of times.

While loops

While loops run until a condition is complete.

int a = 1
while(a < 10) { //runs 9 times
    println(a)
    a++
}

It is common to use while loops for boolean values such as running.

For loops

For loops run for a certain amount of times or for each element in an array. A for loop uses in to determine the range.

To iterate over a range, use the range expression ...

for (i in 1..4) {
    println(i)
}

You can also use downTo and step for a more customized range.

for (i in 6 downTo 0 step 2) {
    println(i)
}

When you have a collection of elements, it is often easier to use a foreach loop. A foreach loop will only allow read access to an item in a list, and will not provide an index.

for (item in list) println(item)

To iterate through an array with the index, use the indices property of an array.

for (i in array.indices) {
    print(array[i])
}

When statements

A when statements replaces the switch operator of many C based languages. When statements iterate through the various cases defined by you and has a default in case there is no case to match.

Here is an example of a switch statement to determine odd or even numbers:

when(x) {
    x.isOdd() -> println("x is odd")
    x.isEven() -> println("x is even")
    else -> println("x is not odd or even")
}

You can use ranges, is and !is, inequalities and raw values as potential cases.

Collections

Collections are important in any language, and go by many names (array, list etc). Kotlin collections are created by using listOf.

val colors = listOf("red", "blue", "orange");

Collections can be access using [].

print(colors[0]) //prints "red"

Collections can also be filtered using .filter().

colors.filter { it.startsWith("b") }

The variable it refers to each element (an internal foreach loop).
